Part III

Installing Oracle Clusterware and Oracle Real Application Clusters

Part III describes the two-phase installation process of how to install Oracle Clusterware and Oracle Database 10g with Real Application Clusters (RAC). It also explains how to create RAC databases, and it describes the post-installation tasks. The chapters in Part III are:

  • Chapter 4, "Installing Oracle Clusterware on Windows-Based Systems"

  • Chapter 5, "Installing Oracle Database 10g with Real Application Clusters"

  • Chapter 6, "Creating RAC Databases with the Database Configuration Assistant"

  • Chapter 7, "Real Application Clusters Post-Installation Procedures"
