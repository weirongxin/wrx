This screenshot includes fields and options that enable you to define a disk group. Included is a table of disks. In this screenshot, all discovered disks are displayed, included disks that belong to other disk groups. A drop-down list enables you to change the list so that it displays only candidate disks. For each disk, the following fields are included: disk path, header status, ASM disk name, size, and failure group. A checkbox to the left of each disk enables you to include the disk in the disk group being created.
